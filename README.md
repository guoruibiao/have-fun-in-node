## 糗事百科小爬虫

用到了一些第三方模块，外加对于`events`模块的消息监听的使用。

- `superagent`: 一款网页爬取库，很好用。
- `cheerio`: 一款网页解析库，内置`htmlparser`支持，类似`JQuery`的选择器，对于回调函数写起来确实非常的方便。
- `cli-color`: 一款命令行下的多彩终端助手，支持多种颜色， 多种样式的显示。
